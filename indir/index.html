Milis Linux 2.1 Sürümleri

Milis Linux 2.1 için beta sürüm çalışmaları devam etmektedir. Bu sürüm öntanımlı olarak Wayland grafik sunucusu kullanmaktadır.

  • Milis 2.1 Masaüstü Alfa (Temmuz 2021) - shasum

Giriş Bilgileri

giriş : mls / şifre : mls

Milis Linux'u ilk açtığınızda ağ bağlantınızı yaptıktan sonra terminal uygulamasını açıp:

mps gun

komutunu verdiğinizde son güncel değişiklikler çekilecektir. Kurulum için uçbirim(terminal) uygulamasını açarak aşağıdaki komut ile kurulumu başlatabilirsiniz:

milis-kur

Milis Linux 2.0 Sürümleri

Milis Linux 2.0 için yapılan çalışmalar kararlı sisteme yakın minimal ve XFCE4 beta imajlarından oluşmaktadır. Milis 2.1 sürümünden farklı olarak X grafik sunucusunu kullanılmaktadır.

  • Milis 2.0 Xfce4 Masaüstü Beta (Kasım 2020) - shasum
  • Milis 2.0 Minimal Beta (Kasım 2020) - shasum

Giriş Bilgileri

giriş : root / şifre : toor

Milis Linux'u ilk açtığınızda ağ bağlantınızı yaptıktan sonra terminal uygulamasını açıp:

mps gun -H

komutunu verdiğinizde son güncel değişiklikler çekilecektir. Kurulum için uçbirim(terminal) uygulamasını açarak aşağıdaki komut ile kurulumu başlatabilirsiniz:

milis-kur
